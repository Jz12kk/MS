4.1.1. Curso Normal e Alternativo - Cliente

4.1.1.1. Efetuar Cadastro

1. O Cliente carrega a pagina de cadastro.
2. O Sistema retorna a pagina de cadastro.
3. O Cliente insere os dados para cadastro.
4. O Sistema verifica os dados inseridos.
4.1. O Sistema analisa se houve algum erro.
4.2. O Sistema retorna a mensagem de erro.
5. O Sistema grava os dados no banco de dados.
6. O Sistema encaminha o Cliente cadastrado para a pagina inicial.
7. O Sistema informa que o cadastro foi realizado com sucesso.

4.1.1.2. Agendar Visita

1. O Cliente carrega a pagina de visitas.
2. O Sistema verifica dados do Cliente.
2.1. Caso Cliente nao esteja logado retorna para a pagina inicial.
3. O Sistema retorna a pagina de visita.
4. O Cliente agenda horario de visitas.
5. O Sistema verifica os dados da visita.
5.1. O Sistema analisa se houve algum erro.
5.2. O Sistema retorna a mensagem de erro.
6. O Sistema grava os dados no banco de dados.
7. O Sistema encaminha o Cliente cadastrado para a pagina inicial.
8. O Sistema informa que a visita foi agendada com sucesso.

4.1.1.3. Efetuar Login

1. O Cliente carrega a pagina de login.
2. O Sistema retorna a pagina de login.
3. O Cliente insere os dados para login.
4. O Sistema verifica os dados inseridos.
4.1. O Sistema analisa se houve algum erro.
4.2. O Sistema retorna a mensagem de erro.
5. O Sistema retorna os dados do banco de dados.
6. O Sistema encaminha o Cliente logado para a pagina inicial.
7. O Sistema informa que o login foi realizado com sucesso.

4.1.2. Curso Normal e Alternativo - Admin

4.1.2.1. Gerenciar Casa

1. O Admin carrega a pagina de gerenciamento de casas.
2. O Sistema retorna a pagina de gerenciamento de casas.
3. O Admin escolhe uma opção.
4. O Sistema verifica a opção escolhida.
4.1. Alterar Casa foi escolhido.
4.1.1. O Admin escolhe a casa para ser alterada.
4.1.2. O Admin insere os dados para serem alterados.
4.1.2.1. O Sistema analisa se houve algum erro.
4.1.2.2. O Sistema retorna a mensagem de erro.
4.1.3. O Sistema grava os dados no banco de dados.
4.2. Deleta Casa foi escolhido.
4.2.1. O Admin escolhe a casa para ser deletada.
4.2.1.1. O Sistema analisa se houve algum erro.
4.2.1.2. O Sistema retorna a mensagem de erro.
4.2.2. O Sistema desativa a casa informada no banco de dados.
4.3. Adicionar Casa foi escolhido.
4.1.1. O Admin insere os dados da casa para serem registrados.
4.1.1.1. O Sistema analisa se houve algum erro.
4.1.1.2. O Sistema retorna a mensagem de erro.
4.1.2. O Sistema grava os dados no banco de dados.
5. O Sistema informa que os dados foram alterados com sucesso.

4.1.2.2. Cadastrar Admin

1. O Admin carrega a página de admin.
2. O Sistema retorna a página de admin.
3. O Admin seleciona o Usuário que terá o Nível de Acesso modificado.
3.1 O Sistema analisa se houve algum erro.
3.2 O Sistema retorna a mensagem de erro.
4. O Sistema grava os dados alterados no banco de dados.
5. O Sistema informa que os dados foram alterados com sucesso.
